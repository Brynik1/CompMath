Что, в целом, происходит?
Программке (на питончике!) даются наблюдения за траекториями вершин (с запаздыванием и погрешностью), программка восстанавливает все что может  

- Восстанавливает размер куба
- По траекториям вершин восстанавливает наблюдаемую траекторию центра масс
- по наблюдаемой траектории центра масс восстанавливает изначальное состояние ценрта масс + фулл траектори центра масс (гаусс ньютон)
- Восстанавливает траекторию всех вершин до наблюдений  
- Восстанавливает все циферки для тела (траекторию, скорость в каждой точке, кватернион в каждой точке, угловой момент в каждой точке)  

Плюс визуализация с 5 режимами (и свободной камерой):  
1) Траектории наблюдений за вершинами  
2) Траектории центра масс (истинная и восстановленная)  
3) Траектории вершин (истинные и восстановленные)  
4) Пустое поле чтобы с кубиком играться  
key k: запуск летящего кубика
